# asset-router

## Dev with Foundry
- install [foundry](https://book.getfoundry.sh/getting-started/installation#installation) if you haven't.

- install deps
```
foundryup
```

- build
```
forge build
```

- test
```
forge test
```

## Deploy with Hardhat
- install deps
```
yarn
```

- build
```
yarn build
```

- run deploy script
```
yarn deploy --network mandala
```
